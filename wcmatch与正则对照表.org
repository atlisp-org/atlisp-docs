#+title: wcmatch 用法

* wcmatch 与正则表达式对比
| wcmatch | preg          | 意义                                     |
|---------+---------------+------------------------------------------|
| #       | \d 或 [0-9]   | 数字                                     |
| @       | \D 或[a-zA-Z] | 字母                                     |
| .       | [\W_ ]        | 非字母数字的符号                         |
| *       | [\w\W]*       | 所有字符                                 |
| ?       | [.\n\t]       | 字母、数字、符号、空格等                 |
| ~...    | [^...]        | 不包含                                   |
| [...]   | [...]         | ...中的任意一个                          |
| [~...]  | [^...]        |                                          |
| -       | -             | 用于[]中表示范围如[0-9][a-z]             |
| ,       | (｜)          | 或                                       |
| `       | \             | 转义，将有匹配意义的字符作为普通字符对待 |



* wcmatch 中的等价表示
| # | [0-9]    |
| @ | [a-zA-Z] |
| . | [~#@]    |
| ? | [.#@]    |
